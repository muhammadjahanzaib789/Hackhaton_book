## The 7 Pipeline Stages

1. **Voice Input** - Capture spoken commands
2. **Intent Understanding** - Parse user intent
3. **Task Decomposition** - Break into subtasks
4. **Path Planning** - Plan navigation route
5. **Navigation** - Execute movement
6. **Object Detection** - Identify targets
7. **Manipulation** - Physical interaction
